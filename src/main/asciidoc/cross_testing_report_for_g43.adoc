== Sonstiges
* Optik der Anwendung
 - gute freundliche Aufmachung
 - einfaches überschaubares Design
 
 - Zufallsfarben bei Parzellenbelegung verwirren
 
* Fehlende Features
- Nutzer hinzufügen
- Nutzer löschen
- Vorstand kann Bewerbungen bearbeiten, annehmen, löschen und ablehnen

* Interaktion mit der Anwendung (Usability)
 - Rollup für Parzellendetails ist mit sehr viel Geklicke verbunden
 - Nach dem Logout muss man Cookies zustimmen…
 
 *Sonstiges Sonstiges
 - durch die Dummy-Beschwerden kann der Wassermann auf fremde Wasserstände zugreifen (z.B. Beschwerden --> Plot 2)

== Verbesserungsvorschläge
* Was kann noch weiter verbessert werden?

- Responsiveness für kleinere Displays
- News: Beitragsverfasser anzeigen
- Terminbearbeitung: Zeit und Datum änderbar machen
- "Mitglied werden" Button im Login-Bereich bzw. Startseite
- [QoL][/anlage] Übersichtslegende - für bessere Sichtbarkeit - über Tabelle legen
